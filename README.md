# INT1303 - An toàn và bảo mật hệ thống thông tin

## Giới thiệu
Repo này được tạo cho môn học **INT1303 - An toàn và bảo mật hệ thống thông tin**.  
Nội dung bao gồm các tài liệu, bài tập lớn và báo cáo liên quan đến chủ đề bảo mật hệ thống thông tin.

## Nội dung repo
Repo hiện có các file chính:
- **ATBMHTTT - D22G0803.docx** → Bản báo cáo chi tiết bài tập lớn.
- **ATBMHTTT - D22G0803.pdf** → Phiên bản PDF của báo cáo, thuận tiện để đọc và chia sẻ.
- **ATBMHTTT - D22G0803.pptx** → Slide thuyết trình cho bài tập lớn.
- **README.md** → Tài liệu mô tả repo.

## Mục tiêu
- Tìm hiểu các khái niệm cơ bản về **an toàn và bảo mật thông tin**.
- Phân tích các mối đe dọa, lỗ hổng và biện pháp phòng chống.
